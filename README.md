# The GAP package fgab

This package is developed for computation problems of `FGAb`, i.e. the abbreviation of finitely generated abelian groups. For example, we are able to compute the kernel and cokernel of a `FGAbHomomorphism`, along with the direct sum, we can compute all finite limits and colimits, in particular, pullback and pushout.
